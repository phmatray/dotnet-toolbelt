> A lightweight collection of simple .NET helper utilities distributed as a NuGet package.

## Description
simple-helpers is a lightweight .NET utility library providing commonly needed helper classes and extension methods. It is packaged as a NuGet package for easy integration into any .NET project.

## Features
- Common helper classes and extension methods
- Lightweight and dependency-free
- NuGet package distribution

## Getting Started
```bash
dotnet add package SimpleHelpers
```
