# ccross-helper

> A collection of portable and specialized C# helper classes to accelerate cross-platform development.

## Description
ccross-helper is a utility library providing a set of reusable portable and specialized C# classes designed to simplify and accelerate cross-platform .NET development. It covers common patterns and helpers used across multiple projects.

## Features
- Portable helper classes for cross-platform development
- Common utilities and extension methods
- Lightweight and easy to integrate

## Getting Started
```bash
dotnet add package phmatray.ccross-helper
```
